### Development requirements:

- Node.js
- [Yarn](http://yarnpkg.com)
- [Now CLI](https://github.com/zeit/now-cli)

## Development

### Install dependencies
```bash
yarn global add now # install Now CLI
yarn install # install dependencies
```
### Dev server
```bash
yarn run dev
```

## Production deploy

### Build project
```bash
yarn run build
```

### Deploy to Now
```bash
yarn run deploy
```
